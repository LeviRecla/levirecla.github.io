Modernizing Legacy Scientific Code

Excellence in Undergraduate Research and Creative Activity Award

This project involved manually translating approximately 20,000 lines of legacy Fortran code into efficient, modular Python. The original code was used for self-absorption correction in Extended X-ray Absorption Fine Structure (EXAFS) analysis â€” a critical process in materials characterization and spectroscopy research.

Context & Motivation

The legacy Fortran routines originated from RSXAP, a research toolkit that lacked maintainability and integration with modern scientific workflows. I sought to preserve its scientific integrity while making it accessible to contemporary researchers using Python.

Challenges

  • Deciphering nested subroutines, COMMON blocks, and implicit variable declarations in Fortran
  • Ensuring numerical accuracy through test-driven translation
  • Maintaining the performance needed for handling real-world EXAFS datasets

Results & Impact

The completed Python implementation retains scientific correctness while improving readability, modularity, and integration with tools like NumPy and SciPy. It laid the groundwork for a larger research initiative focused on automating Fortran-to-Python code translation using language models and fine-tuning techniques.

This project was presented at the 2025 Boise State Undergraduate Research Showcase and archived on ScholarWorks.
